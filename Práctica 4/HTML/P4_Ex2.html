Las aplicaciones web

Las aplicaciones web son aplicaciones a las que se accede mediante un navegador y están alojadas en servidores dentro de una Intranet o en Internet.

Las principales ventajas de aplicaciones web de escritorio son:

  • Se pueden ejecutar desde cualquier equipo informático siempre que tenga conexión a Internet o Intranet.
  • No es necesario utilizar un sistema operativo en concreto, cualquiera sirve.
  • No hay que instalar ningún programa, con el navegador es suficiente.
  • No hay que preocuparse del espacio ocupado por los datos, de eso se encarga el servidor.

Una aplicación web de escritorio debería tener formato web y ofrecer alguna de estas prestaciones:

  1. Gestión de agendas: tomar notas sobre las cosas que hay que hacer.
  2. Gestión de calendarios: mostrar todos los dias del año, por semanas, meses o dias.
  3. Gestión de citas: señalar una hora y un día para que tenga lugar un encuentro.
